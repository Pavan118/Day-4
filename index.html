VS Code-Integrated Development Environment(IDE)

What is VS code?

Visual Studio Code(VS Code) is a free, open-source code editor developed by Microsoft. It is lightweight, highly customizable, and supports a wide range of programming languages and frameworks, making it popular among developers for various types of software development. VS Code is designed to be fast, efficient, and packed with features that boost productivity.

Features of VS code:

  • IntelliSense: A set of features that provide code completion, parameter info, quick info, and member lists. This can help improve productivity by reducing the amount of code a developer needs to write manually
  • Version control: VS Code integrates with Git, which allows users to collaborate with others and keep track of code changes.
  • Syntax highlighting: Helps users be instantly productive with syntax highlighting, bracket-matching, auto-indentation, and box-selection.
  • Debugging: VS Code includes built-in debugging features.
  • Code navigation: VS Code includes code navigation features.
  • Extensions: Users can install extensions to add functionality.
  • Customization: Users can change the theme, keyboard shortcuts, and preferences.

HTML Emmets:

Emmet is a powerful toolkit integrated into VS Code that helps you write HTML, CSS, and other code faster by expanding abbreviations into full code snippets. Emmet allows developers to write complex code in a very short, shorthand form and then expand it with just a key press (usually Tab).

Emmet Symbol Description
! Represent basic structure of Html.
> Creates a child element.
+ Creates sibling elements next to each other.
* Repeats elements.
() Groups elements together.
[] Adds attributes to elements.
{} Adds text inside an element.
$ Adds numbers to repeated elements.

ShortcutKeys for VS Code:

Shortcut keys in VS Code (Visual Studio Code) are key combinations that allow you to quickly perform actions and commands without navigating through the menus.

Shortcut Key Description
ctrl+shift+P Open Command Palette
ctrl+S Save the File
ctrl+N Opens the New file
ctrl+O Open the file
ctrl+Z Undo
ctrl+Y Redo
ctrl+A Select All
ctrl+shift+E Show Explorer
